# 🗒️ Нота — умный ассистент для заметок и напоминаний

Telegram-бот, который **анализирует фото, расшифровывает голосовые и понимает текст**
через ИИ (Google Gemini), а потом сам предлагает: сохранить заметку или поставить
напоминание — с учётом дедлайнов.

Характер бота: спокойный и дотошный к деталям, но бодро подгоняет, когда дедлайн близко. 🙂

---

## ✨ Возможности

- 📷 **Анализ фото** — квитанции, списки задач, расписания, доски. ИИ извлекает суть, суммы, даты.
- 🎤 **Голосовые → текст** — наговори, бот расшифрует и разберёт.
- ✍️ **Умный разбор текста** — «напомни завтра в 18:00 позвонить маме» → готовое напоминание.
- ⏰ **Контекстные напоминания** — с деталями, с умными предложениями времени (за день до дедлайна и т.п.).
- 🗂️ **Заметки** — хранение, категории, поиск.
- 💎 **Тарифы** — FREE / PRO / PREMIUM (задел под платежи).

## 🔑 Где взять ключи

### 1. Токен Telegram-бота
1. Открой [@BotFather](https://t.me/BotFather) в Telegram.
2. `/newbot` → задай имя и username.
3. Скопируй токен вида `1234567890:AAE...` → это `BOT_TOKEN`.

### 2. Ключи Gemini (бесплатно) + ротация
1. Зайди на **https://aistudio.google.com/app/apikey** (нужен Google-аккаунт).
2. Нажми **Create API key** → скопируй.
3. Для нескольких пользователей создай **3–5 ключей** и впиши их через запятую в
   `GEMINI_API_KEYS`. Бот будет чередовать их и **автоматически переключаться на
   следующий, когда ключ упрётся в лимит** (429).

> ⚠️ Ключи одного Google-проекта делят **общую** квоту. Чтобы лимиты реально
> суммировались — бери ключи из **разных Google-аккаунтов/проектов**.
> Для 20–50 человек хватает 3–5 ключей. При росте проще включить
> платный тариф Gemini на одном ключе (стоит копейки за запрос).

## 🚀 Запуск локально (Windows / PowerShell)

```powershell
# 1. Перейти в папку проекта
cd "C:\Users\Матвей\Desktop\бот напоминалка"

# 2. Создать и активировать виртуальное окружение
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 3. Установить зависимости
pip install -r requirements.txt

# 4. Создать .env из примера и заполнить ключи
Copy-Item .env.example .env
notepad .env        # вставь BOT_TOKEN и GEMINI_API_KEY

# 5. Запустить
python main.py
```

После запуска напиши боту в Telegram `/start` и пришли фото/голос/текст.

> Если PowerShell ругается на активацию окружения, выполни один раз:
> `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned`

### Команды для теста тарифа
`/setplan pro` — переключить свой тариф (доступно, пока `ADMIN_IDS` пуст, либо если ты в списке админов).

## ☁️ Деплой на Railway

1. Залей проект в репозиторий GitHub (`git init`, `git add .`, `git commit`, `git push`).
2. На [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo**.
3. **Variables** — добавь переменные окружения (см. таблицу ниже).
4. **Settings → Start Command**: `python main.py` (или Railway сам возьмёт `worker` из `Procfile`).
5. ⚠️ **База данных (важно!)** — файловая система Railway сбрасывается при редеплое.
   Чтобы заметки/напоминания не терялись:
   - **Settings → Volumes → New Volume**, точка монтирования, например `/data`.
   - Поставь переменную `DATABASE_PATH=/data/bot.db`.
6. Готово — бот запустится и будет работать через polling (внешний URL/порт не нужен).

### Переменные окружения (Railway → Variables)

| Переменная | Значение |
|---|---|
| `BOT_TOKEN` | токен от BotFather |
| `GEMINI_API_KEYS` | ключи через запятую: `key1,key2,key3` (ротация) |
| `GEMINI_MODEL` | `gemini-2.0-flash` |
| `AI_PROVIDER` | `gemini` |
| `TIMEZONE` | `Europe/Moscow` |
| `DATABASE_PATH` | `/data/bot.db` (путь на Volume) |
| `FREE_NOTES_LIMIT` | `50` |
| `ADMIN_IDS` | твой Telegram ID (узнать: [@userinfobot](https://t.me/userinfobot)) |

## 🧱 Структура проекта

```
├── config.py             # настройки из .env

├── .env.example          # шаблон конфигурации

│   ├── base.py           #   абстрактный интерфейс провайдера

│   ├── gigachat.py       #   GigaChat (задел на будущее)

│   └── prompts.py        #   характер бота + промпты
│
├── database/
│   └── db.py             # SQLite (aiosqlite): users, notes, reminders
│
├── scheduler/
│   └── reminders.py      # APScheduler: постановка и срабатывание напоминаний
│
├── bot/
│   ├── personality.py    # тексты и форматирование в характере бота
│   ├── keyboards.py      # inline-кнопки
│   ├── states.py         # FSM-состояния
│   ├── pending.py        # временное хранилище черновиков
│   └── handlers/
│       ├── commands.py   # /start, /notes, /reminders, /sovet, ...
│       ├── photo.py      # обработка фото
│       ├── media.py      # обработка голосовых
│       ├── text.py       # обработка текста
│       ├── intent.py     # общий разбор «намерения»
│       ├── callbacks.py  # нажатия кнопок + ввод своего времени
│       └── common.py     # общие помощники (сохранение/напоминания)
│
└── utils/
    ├── logger.py         # логирование
    ├── tariffs.py        # описание тарифов и лимиты
    └── timeutils.py      # разбор времени и умные предложения
```

## 🔧 Как сменить ИИ-провайдера

В `.env` поставь `AI_PROVIDER=gigachat` (когда GigaChat будет подключён).
Весь остальной код менять не нужно — он работает через общий интерфейс `ai/base.py`.
